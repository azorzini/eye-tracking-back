# Eye-tracking api

## About
Api for [eye tracking webapp](https://github.com/azorzini/eye-tracking-front) that exposes resources for Users, Patients, Clients, Clinic history and appointments

## Features
* CRUD of /users
* CRUD of /patients
* CRUD of /clients
* CRUD of /clinichistory
* CRUD of /appointments

## Tools involved
* Springboot
* Hibernate
* Joda time
* Java 1.8
* Oracle XE 11g db
